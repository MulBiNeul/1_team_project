# WBS (작업 분해 구조표)

| 단계 | 작업 항목 | 세부 내용 | 담당 |
|------|-----------|------------|------|
| 1단계 | 데이터 수집 | 도로교통공단 사고 통계 / 고령자 인구 통계 수집 | 팀원1 |
| 2단계 | 데이터 전처리 | 결측치/이상치 처리, 고령자 기준 필터링 | 팀원1,2 |
| 3단계 | EDA 분석 | 지역별 사고율 분석, 변수 간 시각화 | 팀원2 |
| 4단계 | 피처 설계 | 모델 입력 변수 선정 | 팀원2,3 |
| 5단계 | 모델 구축 | 분류 모델 학습 (RF/XGB 등) | 팀원3 |
| 6단계 | 성능 평가 | Accuracy, F1-score 등 비교 | 팀원3,4 |
| 7단계 | 시각화 | 사고 위험도 지도 (folium, heatmap 등) | 팀원5 |
| 8단계 | 발표자료 작성 | PPT, 보고서, 발표 스크립트 | 팀원4,5 |
| 9단계 | 결과 발표 | 6월 9일 최종 발표 | 전원 |
